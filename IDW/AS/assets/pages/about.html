Made with PokeAPI for ULBRA

This website is populated by a RESTful API called PokéApi, this API covers specifically the video game franchise, providing information on pokémons, movesets, abilities, types, eggs and more. It follows a basic structure of showing a specific pokemon and its stats.

Card Structure

Firstly we have the pokémon sprite, which can be found into in-game pokédex from real pokémon games, followed by the pokemon's name and type(s)

After that we have the pokemon's stats, showing its in-game health, attack and defense points

Modal Card

When you click on a specific pokémon, a modal window will open, this window contains information on the pokémon's:

  • Name
  • Type
  • Sprite
  • Movesets

There are three way to close this window:

  • By clicking on the top right "X" button
  • By pressing the escape key
  • By clicking outside of the modal window
